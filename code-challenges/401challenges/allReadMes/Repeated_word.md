# Repeated Word
This read me contains information about code challenge: Repeated word

## Challenge
Find the first repeated word in a string.

## Approach & Efficiency
findRepeatedWord(String str)
    * Time: O(n)
    * Space: O(n)


## API
  ### Methods:
  RepeatedWord:
 * findRepeatedWord(String str): 
     * Takes in a string, and returns the first repeated word
